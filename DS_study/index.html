import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

data = {'ID': [1, 2, 3, 4, 5], 'Name': ['Raj', 'Priya', 'Amit', 'Neha', 'Rahul'], 'Department': ['IT', 'HR', 'Finance', 'IT', 'HR'], 'Salary': [50000, 45000, np.nan, 155000, 40000], 'Experience': [3, 2, 5, np.nan, 1], 'JoiningDate': ['2019-03-15', '2020-07-01', '2018-08-21', '2019-12-10', '2021-01-12']}
df1 = pd.DataFrame(data)
df1['JoiningDate'] = pd.to_datetime(df1['JoiningDate'])
print("Original DataFrame:\n", df1)

Q1. Fill missing Salary with mean salary.

We calculate the mean of the 'Salary' column and use `fillna()` to replace the missing value.


mean_salary = df1['Salary'].mean()

Q3. Normalize the Experience column using Min-Max scaling.

First, we fill the missing 'Experience' value (e.g., with the median). Then, we apply the Min-Max formula to scale the values between 0 and 1.


df1['Experience'].fillna(df1['Experience'].median(), inplace=True)
min_exp = df1['Experience'].min()
max_exp = df1['Experience'].max()
df1['Experience_Normalized'] = (df1['Experience'] - min_exp) / (max_exp - min_exp)
print("DataFrame with normalized experience:\n", df1[['Name', 'Experience', 'Experience_Normalized']])

Q5. Detect and remove outliers on Salary column.

We use the IQR (Interquartile Range) method. Data points outside 1.5 * IQR below Q1 or above Q3 are considered outliers and removed.


Q1 = df1['Salary'].quantile(0.25)
Q3 = df1['Salary'].quantile(0.75)
IQR = Q3 - Q1
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR
df1_no_outliers = df1[(df1['Salary'] >= lower_bound) & (df1['Salary'] <= upper_bound)]
print("DataFrame after removing salary outliers:\n", df1_no_outliers)
